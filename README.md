# It's Not Magic (Code)

A collection of coding exercises and notebook-based challenges.

## Repository Structure

- `Exercise 1.1/` contains notebooks and challenge folders.
- `exercise_1_1.ipynb` is the exercise notebook.
- `exercise_1_1_solution.ipynb` is the solution notebook.

## Getting Started

1. Create and activate a Python virtual environment.
2. Install Jupyter if needed:

```bash
pip install jupyter
```

3. Launch Jupyter:

```bash
jupyter notebook
```

4. Open the notebook files under `Exercise 1.1/`.

## Notes

- Notebook checkpoints are ignored via `.gitignore`.
- Keep large generated outputs out of version control when possible.
